# 🎓 Personal Learning Goals: Cogs & Queries

This project serves as a structured "Skills Lab" to master a professional full-stack environment. The following goals define what I aim to achieve through this build.

## 🛠️ Technical Mastery Goals

### 1. Database Proficiency (SQL Server)
- **Goal:** Master T-SQL syntax and relational logic.
- **Key Skill:** Designing clean schemas and understanding how to query across multiple tables using `INNER JOIN` and `LEFT JOIN`.
- **Milestone:** Successfully writing the "Master Queries" that act as the game's answer keys.

### 2. Backend Architecture (PHP & PDO)
- **Goal:** Move beyond basic script execution to secure, object-oriented database interaction.
- **Key Skill:** Using **PDO (PHP Data Objects)** to prevent SQL injection and handle database errors gracefully.
- **Milestone:** Building a logic engine that compares two different arrays and returns a boolean "Win" state.

### 3. Frontend Interactivity (jQuery & AJAX)
- **Goal:** Create a "Single Page Application" (SPA) feel.
- **Key Skill:** Using `$.ajax()` to send data to the server and update the UI (DataTables) without a browser refresh.
- **Milestone:** Implementing the "Gear Spin" animation that speeds up during an active AJAX request.

### 4. Dynamic Data Visualization (jQuery DataTables)
- **Goal:** Transform raw JSON data into a professional, searchable interface.
- **Key Skill:** Dynamically generating table headers based on whatever SQL query the user writes.
- **Milestone:** Implementing the `table.destroy()` and re-initialization logic to handle changing data structures.

## 📈 Professional Growth Goals

*   **Project Completion:** Taking a concept from a "Project Charter" to a functional, documented repository.
*   **Problem Solving:** Debugging the "handshake" between the SQL Server's specific data types and PHP's array handling.
*   **UI/UX Design:** Using **Bootstrap 5** to create a themed, dark-mode "Industrial" aesthetic that is intuitive for a first-time user.
